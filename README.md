## 书籍

一个图书条目包含三个信息，书名、链接、ISBN。

- 书名: 包括标题和副标题，不加上第几版。

- 链接: 使用doi，其次是出版社官网，再次是亚马逊和豆瓣，最次是一个相对官方稳定的网站。

- ISBN: 优先考虑13位ISBN，和eISBN。
